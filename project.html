Youtube Replica

A website made with HTML and CSS

The project is a simple HTML and CSS implementation of a YouTube web page wireframe. It features a header with the YouTube logo, a side navigation bar on the left with links to different sections, and a main content area displaying video thumbnails, titles, and channel names. The wireframe provides a basic visual representation of how the YouTube page layout is structured, but it lacks actual functionality and dynamic elements found in the real YouTube website.

Text Based Pokemon Battle

A program made with JavaScript, HTML, CSS

In this text-based Pokemon battle prototype, you'll experience intense encounters with wild Pokemon. Engage in turn-based battles by clicking on three available buttons: (Attack) to deal damage, (Switch) to change your active Pokemon, or (Run) to attempt an escape. To win you must bring the other pokemon's health to 0.

Google Replica

A website made with HTML and CSS

In this basic representation of the Google homepage using HTML and CSS. It features a top navigation bar with links for About and Store, as well as a Google logo, search input box, and "Google Search" and "I'm Feeling Lucky" buttons. The design is kept simple and includes responsive elements for a cohesive user experience on various devices. However, this is a simplified version and lacks the full functionality and complexity of the actual Google homepage.
